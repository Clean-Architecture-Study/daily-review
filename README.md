### Rules

- 코드 업로드 담당 : 월 - 송연, 화 - 우혁, 수 - 규림, 목 - 동욱 
- 코드 업로드 : 다음날 수업 전까지 업로드
<br><br>


### Convention Rule

**Issue** | `[스터디 주차] 기능`  
ex) [Week1] 이매일 인증, 결제

**PR** | `[스터디 주차] #이슈번호 Issue 제목`  
ex) [Week1] #1 이메일 인증, 결제

**Commit** | `feat: 기능 구현 내용`<br> 
ex) feat: 이메일 인증

**Branch** | `스터디 주차/기능` <br>
ex) week1/payment
